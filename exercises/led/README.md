This folder contains all the exercises focused on how LEDs work! This step is crucial and should be the very first thing you study before moving forward.

### Things you should know before proceeding:
* An **LED** stands for **Light Emitting Diode**. It is a semiconductor device (a diode) that emits light when an electrical current passes through it.
* **Anode and Cathode:** Every LED has an anode (positive) and a cathode (negative) side, so be careful with its orientation when connecting it to any input or output!
* **USB Power Rail:** On the Arduino Uno, when powered via USB, the board's 5V rail provides approximately 5 volts (typically fluctuating between 4.8V and 5.1V).
* **Digital Pins:** Digital pins (D0 to D13) provide a logic level of around 5V when configured as `OUTPUT` and set to `HIGH`.
* **Short Circuit Protection:** You must always use a resistor to avoid causing a short circuit by connecting 5V directly to GND!
